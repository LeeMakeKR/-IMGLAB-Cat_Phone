# 📞 Cat_Phone

복고풍 전화기를 이용한 인터랙티브 오디오 설치 작품입니다.  
수화기를 들면 랜덤 MP3 음성이 재생되고, 내려놓으면 자동으로 벨이 울립니다.

---

## 하드웨어 구성

| 부품 | 역할 |
|------|------|
| Arduino UNO | 메인 컨트롤러 |
| VS1053 + SD 카드 모듈 | MP3 디코더 및 재생 |
| L298N | 모터 드라이버 (벨 모터 제어) |
| PAM8406 | 오디오 앰프 |
| 수화기 훅 스위치 | 수화기 들림/내려놓음 감지 |

---

## 핀 배선

| 핀 번호 | 연결 |
|--------|------|
| 10 | 수화기 훅 스위치 (⚠️ SPI SS 충돌 주의 — 변경 권장) |
| 3 | L298N IN1 (모터) |
| 4 | L298N IN2 (모터) |
| 5 | L298N ENA (모터 속도 PWM) |
| 9 | SD 카드 CS |
| 11 | SPI MOSI |
| 12 | SPI MISO |
| 13 | SPI SCK |

> ⚠️ 핀 10은 Arduino UNO의 SPI SS 핀으로, 버튼 연결 시 충돌이 발생합니다.  
> 버튼은 핀 2, 6, 7, 8, A1 등 SPI 비관련 핀으로 변경을 권장합니다.

---

## 동작 흐름

```
[수화기 내려놓음 (HIGH)]
  └─ 5분 대기
      └─ 벨 모터 패턴 시작 (2초 ON → 3초 OFF, 최대 10초)
          └─ 10초 경과 → 다시 5분 대기 반복

[수화기 들어올림 (LOW)]
  └─ 모터 즉시 정지
      └─ 1초 대기 후 track001~004 중 랜덤 1곡 1회 재생

[재생 중 수화기 내려놓음 (HIGH)]
  └─ MP3 정지 → 5분 타이머 리셋 → 반복
```

## 주요 설정값

`Cat_Phone.ino` 상단에서 조절 가능합니다.

| 변수 | 기본값 | 설명 |
|------|--------|------|
| `WAIT_TIME_MS` | 300000 (5분) | 벨 울리기까지 대기 시간 |
| `RING_DURATION_MS` | 10000 (10초) | 벨 총 울림 시간 |
| `RING_ON_MS` | 2000 (2초) | 벨 패턴 — 울림 시간 |
| `RING_OFF_MS` | 3000 (3초) | 벨 패턴 — 멈춤 시간 |
| `PICKUP_DELAY_MS` | 1000 (1초) | 수화기 들고 MP3 재생까지 지연 |
| `MOTOR_SPEED` | 100 | 모터 속도 (0~255) |
| `MP3_VOLUME` | 2 | MP3 볼륨 (0=최대, 255=최소) |
| `TRACK_COUNT` | 4 | 트랙 파일 개수 (track001~N) |

## SD 카드 및 MP3 파일

- 포맷: **FAT32** (4GB~32GB 권장), 할당 단위 32KB
- 파일 위치: SD 카드 루트 디렉터리
- 파일명 형식: `track001.mp3`, `track002.mp3`, ... , `track004.mp3`

> macOS에서 SD 카드를 다룰 경우 `._trackXXX.mp3` 메타데이터 파일이 생성될 수 있습니다.  
> 이 파일들은 SD 카드에서 삭제하는 것을 권장합니다.
